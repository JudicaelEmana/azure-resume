Hub & Spoke Sécurisé (FortiGate)

Conception d'une architecture réseau pour un groupe bancaire, avec isolation des environnements critiques et filtrage centralisé des flux.

Points clés:

  • 🌐 Architecture 20+ spokes avec filtrage centralisé.
  • ⚡ réduction de 60% des alertes.
  • 🔌 Connectivité hybride via ExpressRoute + VPN (temps d’accès < 50ms).

Image liée au projet

Défis rencontrés:

  • 📈 Optimisation des règles NSG/FortiGate pour éviter les latences.
  • 💸 Gestion des coûts liés au trafic inter-régions.

Axes d'amélioration:

  • 🛡️ Migrer vers Azure Firewall Premium pour un filtrage L7.
  • 🌪️ Intégrer Azure DDoS Protection.

Compétences demontrées:

  • ✅ Réseau Azure
  • ✅ FortiGate
  • ✅ ExpressRoute
  • ✅ NSG
  • ✅ Architecture cloud
